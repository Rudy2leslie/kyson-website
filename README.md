# Kyson Tech - Paper Packaging Website

Official website for Hunan Kyson Tech Co., Ltd. - Professional manufacturer of eco-friendly paper packaging products.

## Website Structure

- **Home** (`index.html`) - Company overview, featured products, certifications
- **Products** (`products.html`) - Complete product catalog with categories
- **Customization** (`customization.html`) - OEM/ODM services and printing options
- **About Us** (`about.html`) - Company history, factory, certifications
- **Contact** (`contact.html`) - Inquiry form and contact information

## Features

- Responsive design for all devices
- Product catalog with filtering
- Contact form with FormSubmit integration
- SEO optimized
- Fast loading with CDN

## Technologies

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts
